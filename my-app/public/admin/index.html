AESA Pages CMS

Use Pages CMS to edit the AESA website content and upload images through GitHub. Changes are saved to rmitaesa/AESA, then GitHub Pages rebuilds the live website automatically.

Future committee members should sign in with the shared RMIT AESA GitHub account, or an approved GitHub account that has write access to the repository.
  1. Open Pages CMS and sign in with GitHub.
  2. Select the rmitaesa/AESA repository.
  3. Edit the section you need, save, then wait for GitHub Pages to deploy.
