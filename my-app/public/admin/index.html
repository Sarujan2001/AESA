AESA Pages CMS

Use Pages CMS to edit the AESA website content through GitHub. Changes are saved to rmitaesa/AESA, then the live website rebuilds automatically.

Sign in with the approved RMIT AESA GitHub account. Do not use a personal account unless the committee has intentionally given it write access. And if you not intend to edit the website, please do not sign in to Pages CMS at all. This is to prevent any accidental changes and to keep the repository secure.
  1. Click the button below.
  2. Sign in with GitHub when Pages CMS asks.
  3. Select the rmitaesa/AESA repository.
  4. Edit the section you need, save, and wait a few minutes for the website to update.
